- `主页.html` - 主页面，包含完整的登录/注册模态框和导航系统

## 功能特性

### 1. 页面导航系统
- **主页 → 活动页面**: 点击导航栏中的"活动"按钮
- **活动页面 → 主页**: 点击导航栏中的"首页"按钮
- 当前页面在导航栏中会有深色圆角背景高亮显示

### 2. 登录/注册模态框
- 完整的登录和注册表单
- 平滑的切换动画
- 响应式设计，支持桌面和移动端
- 支持ESC键关闭

### 3. 响应式设计
- 移动端友好的汉堡菜单
- 自适应布局
- 优化的触摸交互

## 使用方法

1. **访问页面**：
   - 主页：`http://localhost:8000/主页.html`
   - 活动页面：`http://localhost:8000/活动页面.html`

2. **测试功能**：
   - 点击"登录"或"注册"按钮测试模态框
   - 使用导航栏在页面间跳转
   - 滚动页面查看所有内容

3. **移动端测试**：
   - 调整浏览器窗口大小
   - 测试汉堡菜单功能

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架
- **Lucide Icons** - 图标库
- **Vanilla JavaScript** - 交互逻辑
- **Google Fonts** - 字体（Inter + Cormorant Garamond）
