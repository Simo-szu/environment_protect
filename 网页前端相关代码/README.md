# YouthLoop 环保网站前端代码

## 📋 项目概述
YouthLoop 是一个专注于环保教育和社区活动的互动网站平台，旨在通过游戏化的方式推广环保理念，鼓励用户参与环保活动。

## 🌟 主要功能

### 核心页面
- **主页** (`主页.html`) - 网站首页，展示平台概览和导航
- **科普页面** (`科普页面.html`) - 环保知识科普和新闻资讯
- **活动页面** (`活动页面.html`) - 环保活动展示和参与平台
- **积分页面** (`积分页面.html`) - 用户积分系统和互动问答

### 用户系统
- **登录页面** (`登录页面.html`) - 用户登录界面
- **注册页面** (`注册页面.html`) - 用户注册入口
- **手机注册** (`手机注册.html`) - 手机号注册方式
- **邮箱注册** (`邮箱注册.html`) - 邮箱注册方式
- **重置密码** (`重置密码.html`) - 密码重置功能

### 个人中心
- **我的页面** (`我的页面.html`) - 个人资料主页
- **修改资料** (`修改资料.html`) - 个人信息编辑
- **我的点赞** (`我的点赞.html`) - 用户点赞记录
- **收藏** (`收藏.html`) - 用户收藏内容
- **回复我的** (`回复我的.html`) - 消息回复管理

### 活动系统
- **活动详情** (`活动详情.html`) - 活动详细信息页面
- **活动报名** (`活动报名.html`) - 活动报名表单
- **活动招募** (`活动招募.html`) - 发布活动招募
- **我的活动中心** (`我的活动中心.html`) - 个人活动管理

### 内容系统
- **科普文章详情** (`科普文章详情.html`) - 环保文章详细页面

### 法律文档
- **用户服务协议** (`用户服务协议.html`) - 平台使用条款
- **隐私政策** (`隐私政策.html`) - 用户隐私保护政策

### 核心脚本
- **用户认证系统** (`user-auth.js`) - 用户登录状态管理和认证功能

## 🎨 设计特色

### 视觉设计
- **统一色彩方案**: 使用环保主题色彩 (#56B949绿色, #F0A32F橙色, #30499B蓝色, #EE4035红色)
- **响应式设计**: 支持移动端和桌面端的完美适配
- **动画效果**: 液体光斑流动动画和落叶飘落效果
- **现代UI**: 使用Tailwind CSS框架，圆角卡片和半透明效果

### 交互体验
- **用户认证**: 完整的登录注册系统，支持头像显示和下拉菜单
- **导航系统**: 统一的导航栏设计，当前页面高亮显示
- **搜索功能**: 全站搜索框，支持知识和活动搜索
- **移动适配**: 汉堡菜单和触摸友好的交互设计

## 🛠️ 技术栈
- **前端框架**: HTML5 + CSS3 + JavaScript
- **样式框架**: Tailwind CSS
- **图标库**: Lucide Icons
- **字体**: Inter (无衬线) + Cormorant Garamond (衬线)
- **动画**: CSS3 Animations + Transitions

## 🚀 本地运行
1. 启动本地服务器: `python -m http.server 8000`
2. 访问: `http://localhost:8000`
3. 主要入口页面: `主页.html`
