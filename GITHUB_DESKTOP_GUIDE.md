# 🖥️ 使用 GitHub Desktop 同步代码（推荐）

## 为什么选择 GitHub Desktop？
- 🎯 图形界面，操作简单
- 🔐 自动处理认证
- 🚀 一键同步
- 📱 适合新手

## 📥 安装步骤

### 1. 下载 GitHub Desktop
访问：https://desktop.github.com/
点击 "Download for macOS"

### 2. 安装并登录
1. 打开下载的 .dmg 文件
2. 拖拽到 Applications 文件夹
3. 打开 GitHub Desktop
4. 使用你的 GitHub 账号登录

### 3. 添加现有仓库
1. 点击 "File" → "Add Local Repository"
2. 选择你的项目文件夹：`environment_protect`
3. 点击 "Add Repository"

### 4. 同步代码
1. 在 GitHub Desktop 中会看到你的更改
2. 在底部输入提交信息（如果需要）
3. 点击 "Commit to main"
4. 点击 "Push origin" 推送到 GitHub

## ✅ 完成！
现在你的代码就同步到 GitHub 了！
