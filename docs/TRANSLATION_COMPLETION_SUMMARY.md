每个页面都遵循一致的结构：
```json
{
  "namespace": {
    "title": "页面标题",
    "subtitle": "页面副标题", 
    "description": "页面描述",
    "badge": "徽章文本",
    "sections": {
      "sectionName": {
        "title": "部分标题",
        "content": "部分内容"
      }
    },
    "actions": {
      "actionName": "操作文本"
    }
  }
}
```
