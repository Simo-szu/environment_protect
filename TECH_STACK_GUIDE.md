## 🎓 新手学习路线图

### 📚 第一阶段：基础知识（1-2个月）

#### 1. HTML/CSS 基础 ⭐
**为什么重要**：网页开发的基石
**学习内容**：
- HTML 标签和语义化
- CSS 选择器、布局（Flexbox、Grid）
- 响应式设计

**推荐资源**：
- [MDN Web 文档](https://developer.mozilla.org/zh-CN/)
- [freeCodeCamp](https://www.freecodecamp.org/)

#### 2. JavaScript 基础 ⭐⭐
**为什么重要**：React 和 TypeScript 的基础
**学习内容**：
- 变量、函数、对象、数组
- ES6+ 语法（箭头函数、解构、模块）
- 异步编程（Promise、async/await）

**推荐资源**：
- [JavaScript.info](https://zh.javascript.info/)
- [阮一峰 ES6 教程](https://es6.ruanyifeng.com/)

#### 3. Git 版本控制 ⭐⭐
**为什么重要**：团队协作必备
**学习内容**：
- Git 基本概念和命令
- 分支管理
- GitHub 使用

**推荐资源**：
- [Pro Git 书籍](https://git-scm.com/book/zh/v2)
- GitHub Desktop（你已经在使用了！）

### 📚 第二阶段：React 生态（2-3个月）

#### 4. React 基础 ⭐⭐⭐
**为什么重要**：项目的核心框架
**学习内容**：
- 组件、Props、State
- 生命周期和 Hooks
- 事件处理

**推荐资源**：
- [React 官方文档](https://react.dev/)
- [React 入门教程](https://zh-hans.reactjs.org/tutorial/tutorial.html)

#### 5. TypeScript ⭐⭐⭐⭐
**为什么重要**：提供类型安全，减少 bug
**学习内容**：
- 基本类型和接口
- 泛型和高级类型
- 与 React 结合使用

**推荐资源**：
- [TypeScript 官方文档](https://www.typescriptlang.org/docs/)
- [TypeScript 入门教程](https://ts.xcatliu.com/)

#### 6. Next.js 框架 ⭐⭐⭐
**为什么重要**：项目使用的全栈框架
**学习内容**：
- App Router（项目使用的路由系统）
- 服务端渲染（SSR）
- API 路由

**推荐资源**：
- [Next.js 官方文档](https://nextjs.org/docs)
- [Next.js 中文文档](https://nextjs.cn/)

### 📚 第三阶段：样式和 UI（1-2个月）

#### 7. Tailwind CSS ⭐⭐
**为什么重要**：项目的样式解决方案
**学习内容**：
- 实用优先的 CSS 框架
- 响应式设计
- 自定义配置

**推荐资源**：
- [Tailwind CSS 官方文档](https://tailwindcss.com/docs)
- [Tailwind CSS 中文文档](https://www.tailwindcss.cn/)

#### 8. Radix UI ⭐⭐
**为什么重要**：项目使用的组件库
**学习内容**：
- 无障碍设计原则
- 组件组合模式
- 自定义样式

**推荐资源**：
- [Radix UI 官方文档](https://www.radix-ui.com/)

### 📚 第四阶段：高级功能（2-3个月）

#### 9. 状态管理 - Zustand ⭐⭐
**为什么重要**：管理应用状态
**学习内容**：
- 全局状态管理
- 状态持久化
- 中间件使用

**推荐资源**：
- [Zustand 官方文档](https://github.com/pmndrs/zustand)

#### 10. 动画 - GSAP ⭐⭐⭐
**为什么重要**：创建流畅的用户体验
**学习内容**：
- 基本动画概念
- 时间轴和序列
- React 集成

**推荐资源**：
- [GSAP 官方文档](https://greensock.com/docs/)

#### 11. 数据可视化 - Recharts ⭐⭐
**为什么重要**：展示环保数据
**学习内容**：
- 图表类型和配置
- 数据格式化
- 交互功能

**推荐资源**：
- [Recharts 官方文档](https://recharts.org/)

## 🚀 快速上手建议

### 对于完全新手：

1. **先从修改现有代码开始**：
   - 修改文字内容
   - 调整颜色和样式
   - 添加简单的组件

2. **使用项目作为学习材料**：
   - 阅读现有代码
   - 理解文件结构
   - 模仿现有模式

3. **循序渐进**：
   - 不要试图一次学会所有技术
   - 专注于一个技术栈，掌握后再学下一个

### 实用学习方法：

1. **边做边学**：
   - 在项目中实践新学的知识
   - 创建小功能来测试理解

2. **阅读官方文档**：
   - 官方文档是最权威的学习资源
   - 跟着官方教程做项目

3. **加入社区**：
   - GitHub 讨论区
   - Stack Overflow
   - 技术博客和论坛

## 💡 学习建议

1. **制定学习计划**：每天 1-2 小时，持续学习
2. **实践为主**：理论结合实际项目
3. **寻求帮助**：遇到问题及时询问
4. **保持耐心**：编程学习需要时间积累

记住：每个开发者都是从新手开始的，关键是保持学习的热情和耐心！
