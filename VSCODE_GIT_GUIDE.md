# 💻 使用 VS Code 同步到 GitHub

## 🚀 VS Code Git 操作指南

### 1. 在 VS Code 中打开项目
```bash
cd environment_protect
code .
```

### 2. 使用 VS Code 的 Git 面板
1. 点击左侧的 Git 图标（分支形状）
2. 你会看到所有更改的文件

### 3. 提交更改
1. 在 "Message" 框中输入提交信息
2. 点击 "✓" 按钮提交

### 4. 推送到 GitHub
1. 点击 "..." 菜单
2. 选择 "Push"
3. 如果提示登录，使用你的 GitHub 账号

### 5. 如果遇到认证问题
1. 按 `Cmd + Shift + P` 打开命令面板
2. 输入 "Git: Clone"
3. VS Code 会引导你完成 GitHub 认证

## 🔧 认证设置
VS Code 会自动处理 GitHub 认证，比命令行简单很多！
