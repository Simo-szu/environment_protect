# YouthLoop 项目研究资料包 ## 传播学+交互设计本科毕业设计报告专用 --- ## A) 项目概述（One-paragraph Summary） YouthLoop（青年环保社交平台）是一个面向青年群体的环保主题社交与教育平台，通过科普内容发布、线下活动组织、社交互动功能和游戏化激励机制，帮助青年人了解环保知识、参与环保活动、养成环保习惯。平台核心价值在于将严肃的环保议题转化为可参与、可量化、可社交的数字体验，通过"寓教于乐、科学严谨、行动导向"的设计理念，降低环保参与门槛，构建青年环保社群。技术架构采用 Next.js（前端）+ Spring Boot（后端）+ PostgreSQL（数据库）的分布式架构，支持多语言（中英文）、响应式设计和暗黑模式。 --- ## B) 产品定位 ### 问题陈述（Problem Statement） 1. **环保知识获取门槛高**：传统环保科普内容专业性强、晦涩难懂，青年群体难以理解和吸收 2. **环保行动参与度低**：缺乏便捷的环保活动发现和报名渠道，线下活动组织分散、信息不对称 3. **环保行为缺乏激励**：个人环保行为难以量化和可视化，缺乏持续参与的动力和反馈机制 4. **环保社群归属感弱**：环保爱好者缺乏交流平台，难以形成社群效应和集体行动力 ### 价值主张（Value Proposition） 1. **知识普及**：提供权威、易懂的环保科普内容（新闻、动态、政策、百科），支持多媒体形式和分类检索 2. **行动赋能**：整合线下环保活动信息，提供一键报名、签到打卡、活动管理等全流程服务 3. **社交互动**：支持评论、点赞、收藏、分享等社交功能，构建环保主题社区 4. **游戏化激励**：通过积分系统、每日任务、签到日历、等级体系和虚拟游戏（深圳低碳规划师）激励用户持续参与 5. **个性化推荐**：基于用户行为和兴趣的智能推荐系统，提升内容发现效率 ### 主要用户群体（Primary User Groups） 1. **环保新手（18-25岁大学生）** - 动机：了解环保知识、完成课程作业、参与校园环保活动 - 需求：简单易懂的科普内容、低门槛的参与方式、社交认同感 2. **环保爱好者（25-35岁职场青年）** - 动机：深度学习环保知识、参与线下活动、结识同好 - 需求：专业深度内容、高质量活动信息、社群归属感 3. **环保行动者（活动组织者/NGO工作者）** - 动机：发布活动信息、招募志愿者、扩大影响力 - 需求：活动发布工具、参与者管理、数据统计分析 4. **游戏化学习者（16-30岁）** - 动机：通过游戏学习环保知识、获得成就感 - 需求：趣味性强的互动体验、即时反馈、可视化进度 --- ## C) 功能清单（Feature Inventory） | 功能名称 | 页面/路由/组件 | 用户目标 | 环保教育目标 | |---------|--------------|---------|------------| | **首页轮播** | /[locale]/page.tsx - **前期**（0-20张卡牌）：资源积累，基础设施建设，每回合抽4张卡 - **中期**（21-45张卡牌）：策略成型，平衡发展，每回合抽3张卡 - **后期**（46-60张卡牌）：冲刺结局，优化指标，每回合抽2张卡 **胜利条件**： - 完成60回合（放置60张卡牌） - 根据最终低碳总分评定结局（4种结局：碳中和先锋、绿色发展、平衡发展、工业优先） ### 学习目标（Learning Objectives） | 游戏机制 | 对应学习目标 | |---------|------------| | **资源平衡** | 理解经济发展与环境保护的权衡关系 | | **卡牌分类** | 认识产业、生态、科技、社会四大领域在低碳城市中的作用 | | **政策卡** | 学习真实的深圳双碳政策（如低碳产业补贴、碳捕捉技术推广） | | **碳交易** | 理解碳排放权交易市场的运作机制 | | **突发事件** | 认识气候变化带来的风险和应对策略 | | **组合技** | 理解系统性思维和协同效应在环保中的重要性 | | **结局评定** | 反思不同发展路径的长期影响 | **具体知识点**： -
