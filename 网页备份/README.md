# YouthLoop 环保主题网站

## 项目概述
YouthLoop是一个以环保为主题的响应式网站项目，旨在通过互动式设计推广环保理念，提供环保知识科普、活动参与和积分奖励系统。

## 本地运行

### 启动服务器
```bash
# 进入项目目录
cd environment_protect/网页前端相关代码

# 启动HTTP服务器
python3 -m http.server 8000

# 访问网站
open http://localhost:8000/主页.html
```

### 页面导航
- **主页**: http://localhost:8000/主页.html
