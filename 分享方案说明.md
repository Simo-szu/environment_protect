
### 完整体验方式 (推荐给技术用户)
```
Windows用户:
1. 解压文件
2. 双击 "启动网站.bat"
3. 浏览器访问 http://localhost:8000/主页.html

Mac/Linux用户:
1. 解压文件
2. 双击 "启动网站.sh" (或在终端运行)
3. 浏览器访问 http://localhost:8000/主页.html
```
